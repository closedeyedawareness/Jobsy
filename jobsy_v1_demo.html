Jobsy NL · V1 Alpha

Paste the job titles people actually use. Jobsy resolves each one to a standard role, then returns its profile and salary range — and flags the ones a human should check.

Titles to match — one per line

Pipeline: Normalize → Exact → Normalized → Synonym → Fuzzy → AI
Exact is case-insensitive. The fuzzy stage is in V1 (real RapidFuzz in the Python build; this demo approximates it).
Reference data is illustrative — the live catalog loads from the Excel library.
